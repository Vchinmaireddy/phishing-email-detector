# Phishing Email Detector

A Machine Learning based Phishing Email Detector that identifies whether an email is legitimate or phishing by analyzing email content, links, keywords, and other suspicious patterns.

---

## Features

* Detects phishing emails using machine learning
* Analyzes:

  * Suspicious keywords
  * Fake URLs and links
  * Sender information
  * Email content patterns
* Real-time prediction system
* User-friendly interface
* High accuracy detection model
* Lightweight and easy to use

---

## Technologies Used

* Python
* Scikit-learn
* Pandas
* NumPy
* Flask / Streamlit (optional for UI)
* HTML, CSS, JavaScript (for frontend)

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/phishing-email-detector.git
```

2. Navigate to the project directory:

```bash
cd phishing-email-detector
```

3. Install required dependencies:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python app.py
```

5. Open the browser and visit:

```bash
http://127.0.0.1:5000
```

---

## How It Works

The system uses Natural Language Processing (NLP) and Machine Learning techniques to classify emails as:

* Legitimate Email
* Phishing Email

The model is trained on labeled datasets containing phishing and safe emails.

### Detection Parameters

* Presence of suspicious words
* Fake or shortened URLs
* Urgent or threatening language
* Sender authenticity
* Excessive use of symbols or attachments

---

## Machine Learning Model

Common algorithms used:

* Naive Bayes
* Logistic Regression
* Random Forest
* Support Vector Machine (SVM)

The best-performing model is saved and used for predictions.

| ------------------------------------------------- | ---------- |
